乐玩自动化测试插件接口说明 for 8.03

GetCode_Ruo_File


函数简介:

若快答题，从文件

 

函数原型:

string  GetCode_Ruo_File (usern,passw,filename,tpye,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

filename 字符串：验证码图片的文件路径（绝对路径或者相对于全局路径的路径）

type 字符串:验证码类型,具体类型和价格请看：http://www.ruokuai.com/home/pricetype

timeout 整形数:答题超时，单位默认为秒，默认60   

 

返回值:

字符串

答题结果|题目ID|错误原因

 

 示例:

 ret = lw.GetCode_Ruo_file ("abc","efg","a.bmp","3040")
