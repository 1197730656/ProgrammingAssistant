乐玩插件接口说明 v7.53

WriteUniversal


函数简介:

对指定地址写入数据

 

int WriteUniversal(addr,v,type)

 

参数定义:
addr  整形数: 要写入的地址。

value   字符串: 要写入数据的字符串形式。

type   整形数: 要写入数据的类型,默认为0，取值方式如下：

                  0：整数型（int）

                  1：短整数型（short ）

                  2:  长整数型（long ）

                  3:  字节型（byte）

                  4:  单精度浮点（float）

                  5:  双精度浮点（double）

                  6:  GBK字符串

                  7:  Unicode字符串

                  8: 字节数组（Array of Bytes 十六进制形式）

                  9: 字节数组（Array of Bytes  十进制形式）

返回值:

整形数:
0 : 失败

1 : 成功

 

示例:

ret = lw.WriteUniversal(1234,“100”,0)

 

注意,在读写之前一定要保证该对象使用过 SetMemoryProcess命令设置过读取的目标进程.参见(SetMemoryProcess.html)之后的所有内存相关命令都一样,不再提示!
