ReadString


函数简介:

读取指定地址的字符串，可以是GBK字符串或者是Unicode字符串.(必须事先知道内存区的字符串编码方式)

 

函数原型:
string ReadString(addr,type,len)

 

参数定义:
addr 整数型:要读取的地址

type 整数型: 字符串类型,0= GBK字符串;1 =Unicode字符串 默认为0

len   整数型:读取长度,需要读取的字节数目.如果为0，则自动判定字符串长度.默认为0

 

 

返回值:

字符串:
成功返回读取到的字符串,失败返回一个空文本.

 

示例:

value =lw.ReadString(12345)
