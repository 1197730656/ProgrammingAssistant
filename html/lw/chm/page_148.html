乐玩插件接口说明 v7.53

ReadUniversal


函数简介:

读取指定地址的数据

 

函数原型:

string  ReadUniversal(addr,type,len)

 

 

参数定义:
addr 整数型:要读取的地址（十进制的绝对地址）

type 整数型:要读取的类型，默认为0,取值方式如下：

                  0：整数型（int）

                  1：短整数型（short ）

                  2:  长整数型（long ）

                  3:  字节型（byte）

                  4:  单精度浮点（float）

                  5:  双精度浮点（double）

                  6:  GBK字符串

                  7:  Unicode字符串

                  8: 字节数组（Array of Bytes 以十六进制形式返回）

                  9: 字节数组（Array of Bytes 以十进制形式返回）

len  整数型: 欲读取长度，读取的类型小于等于5时，此参数无效，6或者7时，如果此参数为0时，自动判断文本的长度，否则读取指定长度，8或者9时，此参数是必须的。默认为0。

返回值:

字符串：

成功返回读取到的数值（以字符串形式返回）

 

示例:

value = lw.ReadUniversal(12345)

MessageBox  value


 

注意,在读写之前一定要保证该对象使用过 SetMemoryProcess命令设置过读取的目标进程.参见(SetMemoryProcess.html)之后的所有内存相关命令都一样,不再提示!
