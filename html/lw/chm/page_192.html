GetCode_Tu_Buffer


函数简介:

超人答题，从缓冲区

 

函数原型:

string  GetCode_Tu_Buffer (usern,passw,buffer,size,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

buffer 整形数：验证码图片的数据所在缓冲区

size 整形数：缓冲区中的数据长度

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 

 示例:

piclen= lw.Capture(0,0,2000,2000,1)

code= lw.GetCode_Tu_Buffer ("abc","efg",lw.x,piclen,"0")

//注意TC要用 lw.x()

 

无需填写验证码类型,平台将会跟据题目自动扣点,价目表请参考page_189.html
