GetCode_Lz_Buffer


函数简介:

联众答题，从缓冲区

 

函数原型:

string  GetCode_Lz_Buffer (usern,passw,buffer,size,type,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

buffer 整形数：验证码图片的数据所在缓冲区

size 整形数：缓冲区中的数据长度

type   字符串：  验证码类型  “0”＝四字英文数字，“23”＝四字中文，请它请查看page_91.html

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 

 示例:

piclen= lw.Capture(0,0,2000,2000,1)

code= lw.GetCode_Lz_Buffer ("abc","efg",lw.x,piclen,"0")

//注意TC要用 lw.x()

 

   注意：如果确定账号密码都没问题，但又连续多次提示账号密码错误，请联系联众的客户，查下看是不是因为多次提供错误的账号密码，导致IP被封。
