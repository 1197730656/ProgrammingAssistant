AdbGetClass


函数简介:

获取应控件的类名

 

函数原型:
int AdbGetClass(item)

 

参数定义:

item  整形数:要取第几个控件的类名,0=获取上次查找控件找到的那个;如果提供一个大于等于1的数值表示,获取第指定个数的控件的类名,在此之前必须保证调用过AdbGetItemCount命令,默认为0.

返回值:
字符串

返回相应控件的类名.

示例:

 

ret=lw.AdbFindUI("标题")

if ret=1 then

    ret =lw.AdbGetClass()

    TracePrint ret

end if
