GetCode_Hao_File

函数简介:

答题，从文件

 

函数原型:

string  GetCode_Hao_File (passw,filename,type,timeout)

 

参数定义:

passw 字符串:密码串,上好爱官网,登录账号查看,好爱官网:http://www.haoi23.net/

filename 字符串：验证码图片的文件路径（绝对路径或者相对于全局路径的路径）

type   字符串： 1004=1-4位纯数字;2004=1-4位纯字母;3004=1-4位数字字母组合;4004=1-4位繁简体中文　其它的到接口说明里看,或者到官方看。

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 示例:

 ret = lw.GetCode_Hao_file ("win12348|123412386123459BA9C","a.bmp","1004")
