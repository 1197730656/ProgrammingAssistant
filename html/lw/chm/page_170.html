乐玩插件接口说明 v7.53

CheckUAC


函数简介:

检测当前系统是否有开启UAC(用户账户控制).

函数原型:

int CheckUAC()

参数定义:

返回值:

整形数:

检测失败返回-1,没开启返回0,已开启返回一个大于等于1的值.

示例:


if lw.CheckUAC() >= 1 then
    TracePrint "当前系统开启了用户账户控制"
end if

注: 只有WIN7 WIN8 VISTA WIN2008以及以上系统才有UAC设置
