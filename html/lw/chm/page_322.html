IniDeleteSection

函数简介:

删除指定配置文件中的指定节。

函数原型:

int IniDeleteSection(file,section)

 

参数定义:

file 字符串:      文件名，指定配置文件的名称，通常以.ini作为文件名后缀。

section 字符串:  欲删除的节的节名称。

 

返回值:

整形数:
0 : 失败
1 : 成功

示例:

// 绝对路径
ret=lw.IniDeleteSection("c:\test_game\cfg.ini", "mSection")


// 相对路径
ret=lw.SetPath ("c:\test_game")
ret=lw.IniDeleteSection("cfg.ini", "mSection")

-----------------------------------------------------------------------------------------------

INI文件结构示例：

//Section＝节名称,用来表示一个段落。
//key=配置项,用来表示一个参数名。
//value=用来存储一个值。
[Section1]

Key1＝value1

Key2＝value2

...

[Section2]

Key1＝value1

Key2＝value2

...

注 : 此函数是多线程安全的. 多线程同时读写同个文件不会造成文件错乱.
