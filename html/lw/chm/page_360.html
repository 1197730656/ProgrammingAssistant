乐玩插件接口说明 v7.812_2

SetUserTimeLimit


函数简介:

设置程序使用的到期时间。

 

函数原型:
int   SetUserTimeLimit(time)

参数定义:

string 字符串 ：到期时间。格式形如[2017年7月26日18时40分]

返回值:

整数型

成功返回1，失败返回0。

 

示例:

ret = lw.SetUserTimeLimit("[2017年10月24日1时41分]")//可以精确到秒，到时间格式不要弄错。

 

注意：

1，本命令在调用的时候需要获取网络时间，如果不能联网，将会一直卡在这个命令。

2，到期后会强制自身进程，同时，删除EXE文件。

3，注意时间格式，如果提供错误格时的时间，将会立即结束自身进程。
