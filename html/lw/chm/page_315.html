GetFindedPicName


函数简介:

取上次调用FindPic命令找到的图片的图片名字。

函数原型:

string GetFindedPicName()

 

参数定义:

返回值:

字符串:

如果上次找到，返回上次找到的图片的图片名字，否则返回一个空文本。

示例:

ret = lw.bindwindow(- 1 , 0, 0, 0, 0, 0, 0)
ret = lw.findpic(0, 0, 2000, 2000, "a.bmp|b.bmp|c.bmp")
