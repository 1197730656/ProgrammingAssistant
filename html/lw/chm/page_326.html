FindBrightPoint


函数简介:

查找指定区域内最亮的点。本函数会设置x,y.

函数原型:

int  FindBrightPoint(x1, y1, x2, y2, lower,Upper)

参数定义:

x1 整形数:区域的左上X坐标
y1 整形数:区域的左上Y坐标
x2 整形数:区域的右下X坐标
y2 整形数:区域的右下Y坐标
lower  整形数:亮度下限，取值范围0－100，0表示查找最亮的点(尽量靠后)。其它值表示当前点的值只要大于等于此参数的值就立即返回（最前）。该参数的默认值是0

Upper   整形数:亮度上限，仅在亮度下限不为0时有效，默认为100。默认为100。

 

返回值:

整形数:
如果找到符合要求的点，返回该点的亮度值。（范围是0－100），否则返回－1。

 

 

示例:
ret = lw.FindBrightPoint(0,0,2000,2000)
If lw.x>= 0 and lw.y>= 0 Then
    MessageBox "找到最亮的点亮度为："&ret
End If

//TC等其它语言要用 lw.x(),lw.y()。

 

 

注意：

HSV色中的V值就是亮度，可用乐玩编程助手查看。
