乐玩插件接口说明 v7.8062

DLConnect


函数简介:

调用本地宽带连接进行ASDL或VPN拨号

 

函数原型:
string  DLConnect(ConnectName,userName,Pwd,url,timeout)

 

参数定义:

ConnectName  字符串：  欲进行拨号的连接名。

userName        字符串：  拨号时调用的账号

Pwd                 字符串：  拨号时调用的密码

url                   字符串：   远程主机域名或者IP，默认为一个空文本，ASDL拨号不需要这个。

timeout           整数型:：拨号超时，默认为60000（一分钟）。－1表示一直连直到连接成功或者失败为止，其它值表示到时间立即返回。

 

 

返回值:

字符串

成功返回一个空文本，失败返回错误原因，格式为 错误代码：错误描述。

 

示例:

 ret = lw.DLConnect(“07221234567”，“myname”,"pwd")
