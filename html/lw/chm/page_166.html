AsmCode


函数简介:

把汇编缓冲区的指令转换为机器码 并用16进制字符串的形式输出

函数原型:

string AsmCode(base_addr)

参数定义:

base_addr 整形数: 用AsmAdd添加到缓冲区的第一条指令所在的地址

返回值:

字符串:
机器码，比如 "aa bb cc"这样的形式

示例:

code = lw.AsmCode(&H405940)
MessageBox code


 

可以在工具的汇编调试里进行测试。
