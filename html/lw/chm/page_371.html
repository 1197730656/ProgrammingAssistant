乐玩自动化测试插件接口说明 for 8.03

GetCode_Cr_Buffer


函数简介:

超人答题，从缓冲区

 

函数原型:

string  GetCode_Cr_Buffer (usern,passw,buffer,size,type,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

buffer 整形数：验证码图片的数据所在缓冲区

size 整形数：缓冲区中的数据长度

type   整形数：  ：0=数字字母混合(当然也包含纯数字,纯字母)　1=数字字母中文混合(当然也包括纯中文)。

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 

 示例:

piclen= lw.Capture(0,0,2000,2000,1)

code= lw.GetCode_Cr_Buffer ("abc","efg",lw.x,piclen,"0")

 

//注意TC要用 lw.x()
