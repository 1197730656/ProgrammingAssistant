IniRead


函数简介:

读取指定配置文件中指定项目的文本内容。

函数原型:

string IniRead(file,section,key,Default)

参数定义:

file 字符串:   文件名，指定配置文件的名称，通常以.ini作为文件名后缀。

section 字符串:   节名称，指定欲读入配置项所处节的名称。

key 字符串:   配置项，指定欲写入配置项所处项目名称。

Default 字符串:   默认文本，指定的条目没有找到时返回的默认值。可空，默认为一个空文本。

返回值:

字符串:
字符串形式表达的读取到的内容

示例:

// 绝对路径
ret = lw.IniRead("c:\test_game\cfg.ini"，“mSection","mkey")

// 相对路径
ret=lw.SetPath ("c:\test_game")
Text = lw.IniRead("cfg.ini","mSection","mkey")

 

 

注 : 此函数是多线程安全的. 多线程同时读写同个文件不会造成文件错乱.
