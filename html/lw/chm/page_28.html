CmpColor


函数简介:

比较指定坐标点(x,y)的颜色

函数原型:

int CmpColor(x,y,color,sim)

参数定义:

x 整形数: X坐标

y 整形数: Y坐标

color 字符串: 颜色字符串,可以支持偏色,多色,例如 "ffffff-202020|000000-000000" 这个表示白色偏色为202020,和黑色偏色为000000.颜色最多支持10种颜色组合. 注意，这里只支持RGB颜色.

sim 小数型: 相似度(0.1-1.0),该参数的缺省值为0.95

 

返回值:

整形数:
0: 颜色匹配
1: 颜色不匹配

示例:

ret = lw.CmpColor(200,300,"000000-000000|ff00ff-101010",0.9)
If ret = 0 Then
    MessageBox "相等"
End If
