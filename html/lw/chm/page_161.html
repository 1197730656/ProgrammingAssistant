乐玩插件接口说明 v7.53

FindCode


函数简介:

特征码查找.

 

函数原型:
int FindCode(code,lose,module,flag)

 

 

参数定义:
code 整形数:要查找的特征码.注明：OD格式的二进制数据。要进行模糊查找的数据十六进制文本,格式如 "558BECB8*????8BE55DC3" ;2个问号代表1字节任何数据(每处问号一定要是双数).星号代表返回内存地址位置,星号有且只能有一个.

lose  整数型: 跳过次数从1开始数,为1或0表示首次找到就返回.

module   字符串: 模块名,要在哪个模块里查,如user32.dll等.如果提供一个空本，则搜索进程内的所有可读内存。默认为一个文本，但建议指定模块名。

flag  整数型: 查找类型,0=要返回找到的地址中的值;1=要返回找到的地址.默认为0

返回值:

整形数:
成功返回找到的值或者地址,失败返回-1

 

示例:

ret = lw.FindCode("558BECB8*????8BE55DC3",0,"run.exe",0 )

 

可以在工具里的死码搜索里进行测试。

 

注意,在读写之前一定要保证该对象使用过 SetMemoryProcess命令设置过读取的目标进程.参见(SetMemoryProcess.html)之后的所有内存相关命令都一样,不再提示!
