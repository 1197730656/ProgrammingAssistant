乐玩插件接口说明 v7.27

GetCursorShape


函数简介:

获取鼠标特征码. 当绑定时,附加信息里有"需要取鼠标特征码"时获取到的是后台鼠标特征，否则是前台鼠标特征.

 

函数原型:

int GetCursorShape(mod)

参数定义:

mod:整数型,获取方式,取值有0和1两种,哪种能取到用哪种,默认值为0.

返回值:

整数型:
成功时，返回鼠标特征码. 
失败时，返回0

示例:


mouse_tz = lw.GetCursorShape()
If mouse_tz = 1235　Then
    MessageBox "找到特征码"
End If

注意,WIN7以及以上系统，必须在字体显示设置里把文字大小调整为默认(100%),否则特征码会变.

注意,在没有调用  binwindow 绑定窗口之前,所有的键盘鼠标命令都无效!!!
