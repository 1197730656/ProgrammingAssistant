乐玩插件接口说明 v7.27

UnRegCom


函数简介:

卸载注册COM的组件,非命令行注册,不容易失败,推荐采用此方法.

 

函数原型:

int  UnRegCom(file)

参数定义:

file 字符串: COM组件的DLL的绝对路径.

返回值:

整形数:
0:失败
1:成功

示例:

 

ret=Plugin.lwEX.UnRegCom ("E:\mycom\lwsoft\lw.dll")

if ret=1 then

     MessageBox "卸载成功"

End

 

注意,这是为按键精灵写的一个扩展插件,使用前要把压缩包里的LwEx.dll放到按键精灵的plugin目录下.
