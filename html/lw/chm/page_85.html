乐玩插件接口说明 v7.30

x

  函数简介:

  同GetX,获取上次找图，找字，找色调用完成后，设置的目标的左上角的X坐标值。

 

  函数原型:

  int X()

 

  返回值:

  整形数:

  返回上次设置的X坐标。

  示例:

  ret=lw.FindPic(0,0,2000,2000,"a.bmp")

 

  if 　lw.x>-1 then

     MessageBox("图片找到了，X坐标为："+lw.x)

  end
