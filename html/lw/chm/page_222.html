乐玩插件接口说明 v7.568

FoobarDrawPic


函数简介:

在指定的Foobar窗口绘制图像

函数原型:

int FoobarDrawPic(hwnd,picName,x1,y1,x2,y2,falg)

参数定义:

hwnd整形数: 指定的Foobar窗口,注意,此句柄必须是通过CreateFoobar函数创建出来的

pic_name字符串: 图像文件名,目前只支持BMP图像.

x1整形数: 左上角X坐标(相对于hwnd客户区坐标)

y1整形数: 左上角Y坐标(相对于hwnd客户区坐标)

x2整形数: 右下角X坐标(相对于hwnd客户区坐标),0=表示x1+图像宽度,默认为0.

y2整形数: 右下角Y坐标(相对于hwnd客户区坐标),0=表示y1+图像高度,默认为0.

flag 整形数:拉伸方式,0=居左上角；1＝填充设置的矩形区域 ；2＝填充至窗口右下角（此时右边，下边参数无效）。默认为0.

返回值:

整形数 :
0 : 失败

1 : 成功

示例:

ret = lw.FoobarDrawPic(foobar,"menu.bmp",0,0)
