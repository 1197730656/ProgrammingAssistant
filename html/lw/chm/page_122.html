GetCode_lz_file


函数简介:

联众答题，从文件

 

函数原型:

string  GetCode_Lz_File (usern,passw,filename,type,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

filename 字符串：验证码图片的文件路径（绝对路径或者相对于全局路径的路径）

type   字符串：  验证码类型  “0”＝四字英文数字，“23”＝四字中文，请它请查看page_91.html

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 

 示例:

 ret = lw.GetCode_lz_file ("abc","efg","a.bmp")

 

   注意：如果确定账号密码都没问题，但又连续多次提示账号密码错误，请联系联众的客户，查下看是不是因为多次提供错误的账号密码，导致IP被封。
