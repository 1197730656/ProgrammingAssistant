GetCode_Hao_Buffer


函数简介:

答题，从缓冲区

函数原型:

string  GetCode_Hao_Buffer (passw,buffer,size,type,timeout)

 

参数定义:

passw 字符串:密码串,上好爱官网,登录账号查看,好爱官网:http://www.haoi23.net/

buffer 整形数：验证码图片的数据所在缓冲区

size 整形数：缓冲区中的数据长度

type   字符串： 1004=1-4位纯数字;2004=1-4位纯字母;3004=1-4位数字字母组合;4004=1-4位繁简体中文　其它的到接口说明里

timeout 整形数:答题超时，单位默认为秒，默认60   

返回值:

字符串

答题结果|题目ID|错误原因

 

 示例:

piclen= lw.Capture(0,0,2000,2000,1)

ret = lw.GetCode_Hao_Buffer("win12348|123412386123459BA9C",lw.x,piclen,"1004")

 

//注意TC要用 lw.x()
