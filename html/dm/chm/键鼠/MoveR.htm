函数简介:

鼠标相对于上次的位置移动rx,ry.    从6.1548版本开始,如果您要使鼠标移动的距离和指定的rx,ry一致,最好配合SetMouseSpeed和EnableMouseAccuracy函数来使用.

函数原型:

long MoveR(rx,ry)

参数定义:

rx 整形数:相对于上次的X偏移
ry 整形数:相对于上次的Y偏移

返回值:

整形数:
0:失败
1:成功

示例:


dm.MoveR rx,ry

 

注: 此函数从6.1550开始，为了兼容某些特殊输入，不再自动设置鼠标的速度和精确度。如果您要使鼠标移动的距离和指定的rx,ry一致,那么最好配合SetMouseSpeed和EnableMouseAccuracy函数来使用

因为rx和ry的偏移量不一定就是鼠标真实的偏移,而是代表了物理鼠标DPI偏移. 如果您需要这个偏移和真实鼠标偏移一致，那么需要如下调用者这两个函数，如下所示:

dm.SetMouseSpeed 10   // 保持移动速度是居中的位置
dm.EnableMouseAccuracy 0 // 关闭精确度开关

当然，这样的设置，可能会对用户造成困扰，比如鼠标操作突然变的很不习惯. 为了兼容考虑，您可以在设置之前,保存这两个信息,然后在您的软件退出时，恢复这些信息即可.
但实际上,关闭精确度和设置为速度居中，是绝大多数人的鼠标最适合的设置 除非一些DPI太低的鼠标，才需要靠这个设置去加速模拟.
