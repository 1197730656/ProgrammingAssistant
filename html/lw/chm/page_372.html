乐玩自动化测试插件接口说明 for 8.03

ReportError_Cr


函数简介:

向打码平台报告打码错误,只在验证码识别成功并且验证码错误时才调用,成功返回1,失败返回0

 

函数原型:
int  ReportError_Cr(usern,passw,id,timeout)

 

参数定义:

usern 字符串:账号

passw 字符串:密码

id       字符串:打码时返回的题目ID.

timeout 整形数:超时，单位默认为秒，默认15

   

返回值:

成功返回1,失败返回0

 

示例:

ret = lw.ReportError_Cr("abc","efg",id)
