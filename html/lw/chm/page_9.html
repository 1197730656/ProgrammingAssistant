和大漠插件的异同

 
 
COM版实现了大漠插件的绝大部分功能，语法和命令名字也几乎一样，但少数命令的调用方式会有所差异。
 
1，找图，找色，找字命令没有intx,inty两个变参指针参数，用来接收返回的左上角坐标，而要在调用相关命令后，再用 GetX()，GetY,GetIdx 等方法方法来接收上次找图，找色，找字命令时，返回的坐标和序号。
     示例：
     ret = lw.FindPic(0,0,2000,2000,"1.bmp|2.bmp|3.bmp","000000",0.9,0)
    If lw.GetX() >= 0 and lw.GetY()>= 0 Then
       MessageBox "找到"
    End If
 
 
 
2，返回的序号是从1开始，而非0.
3，其它差异请看具体说明。
