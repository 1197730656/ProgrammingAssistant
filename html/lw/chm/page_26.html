乐玩插件接口说明 v7.22

FindColor


函数简介:

查找指定区域内的颜色,颜色格式"RRGGBB-DRDGDB",注意,和按键的颜色格式相反 本函数会设置x,y,idx.

函数原型:

int  FindColor(x1, y1, x2, y2, color, sim, dir,timeout,ischick,chickdex,chickdey,chickdelay)

参数定义:

x1 整形数:区域的左上X坐标
y1 整形数:区域的左上Y坐标
x2 整形数:区域的右下X坐标
y2 整形数:区域的右下Y坐标
color 字符串:颜色 格式为"RRGGBB-DRDGDB",比如"123456-000000|aabbcc-202020".注意，这里只支持RGB颜色.
sim 小数型:相似度,取值范围0.1-1.0　该参数的缺省值为0.95
dir 整形数:查找方向 0: 从左到右,从上到下   该参数的缺省值为0
             1: 从左到右,从下到上
             2: 从右到左,从上到下
             3: 从右到左,从下到上
             4：从中心往外查找
             5: 从上到下,从左到右
             6: 从上到下,从右到左
             7: 从下到上,从左到右
             8: 从下到上,从右到左

timeout 整形数:查找超时,单位为毫秒,为0则只找一次,否则一直找,直到找到或者超时为止.  该参数的缺省值为0

ischick　整形数:如果找到是否点击,0=找到不点击;1=如果找到了就点击.该参数的缺省值为0

chickdex　整形数:如果要点击,点击的偏移x坐标.该参数的缺省值为0

chickdey　整形数:如果要点击,点击的偏移y坐标.该参数的缺省值为0

chickdelay　整形数:点击延时,如果找到要点击,找到后等多久才点击.该参数的缺省值为0

返回值:

整形数:
0:没找到
1:找到

示例:


ret = lw.FindColor(0,0,2000,2000,"123456-000000|aabbcc-030303|ddeeff-202020",1.0,0)
If lw.x>= 0 and lw.y>= 0 Then
    MessageBox "找到"
End If

//TC等其它语言要用 lw.x(),lw.y(),lw.idx()
