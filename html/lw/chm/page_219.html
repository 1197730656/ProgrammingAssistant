FoobarSetFont

  函数简介:

  设置指定Foobar窗口的字体信息

  函数原型:

  int FoobarSetFont(hwnd,color,balckColor,size,Format,Bold,Underline,Italic)

  参数定义:

  hwnd整形数: 指定的Foobar窗口句柄,此句柄必须是通过CreateFoobar创建而来

  color整形数: 文本颜色，默认为“000000”（黑色）。

  balckColor 整形数: 文本的背景颜色，-1或者提供一个空文本表明，没有背景色，是透明的.默认为－1

  size整形数: 字体大小，默认为18.

  Format 整形数: 对齐方式，0＝左对齐；1＝横向居中对齐；2＝横向右对齐。默认为0

  Bold 整形数: 是否加粗，0＝不加粗；1＝加精。默认为0

  Underline 整形数: 是否有下划线，0＝没下划线；1＝有下划线。默认为0

  Italic 整形数: 是否斜体，0＝正常；1＝斜体。默认为0

  返回值:

  整形数:
  0 : 失败

  1 : 成功

  示例:

  ret = lw.FoobarSetFont(foobar,"ff0000","-1")
