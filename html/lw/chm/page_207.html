FindNearestPos


函数简介:

根据部分Ex接口的返回值，然后在所有坐标里找出距离指定坐标最近或者最远的的那个坐标.

函数原型:

string FindNearestPos(allPos,x,y,type)

参数定义:

allPos 字符串: 坐标描述串。一般是FindStrEx,FindStrFastEx,FindStrWithFontEx, FindColorEx, FindMultiColorEx,和FindPicEx的返回值.

x 整形数: 横坐标
y 整形数: 纵坐标

type  整形数: 取值方式，0＝取最近点；1＝取最远点。可空，默认为0。

 

返回值:
字符串:
返回的格式是"id,x,y"

 

 

示例:

 ret = lw.FindColorEx(0,0,2000,2000,"aaaaaa-000000",1.0,0)
 ret = lw.FindNearestPos(ret,100,100,10)
 TracePrint ret
