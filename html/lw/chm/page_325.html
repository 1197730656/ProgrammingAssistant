IniEnumSection


函数简介:

枚举指定配置文件中所有的节的节名称。

函数原型:

int IniEnumKey(file)

 

参数定义:

file 字符串:      文件名，指定配置文件的名称，通常以.ini作为文件名后缀。

 

返回值:

字符串:
每个节名用"|"来连接，如果没有节名，则返回空字符串. 比如"aaa|bbb|ccc"

 

示例:

// 绝对路径
ret=lw.IniEnumSection("c:\test_game\cfg.ini")


// 相对路径
ret=lw.SetPath ("c:\test_game")
ret=lw.IIniEnumSection("cfg.ini")

-----------------------------------------------------------------------------------------------

INI文件结构示例：

//Section＝节名称,用来表示一个段落。
//key=配置项,用来表示一个参数名。
//value=用来存储一个值。
[Section1]

Key1＝value1

Key2＝value2

...

[Section2]

Key1＝value1

Key2＝value2

...

注 : 此函数是多线程安全的. 多线程同时读写同个文件不会造成文件错乱.
